# Máster Especialista en Deep Learning con PyTorch

Este repositorio contiene el material de estudio, notas teóricas y la 
implementación de modelos desarrollados durante el 
Máster Especialista de Deep Learning en Python. 
El enfoque principal es el diseño, entrenamiento y despliegue de redes 
neuronales profundas utilizando el framework PyTorch


## Objetivos del Aprendizaje
- End-to-End Pipeline: Desarrollo y evaluación de modelos desde el preprocesamiento hasta la inferencia.
- State-of-the-Art (SOTA): Aplicación de técnicas avanzadas en modelos de última generación.
- Arquitecturas Clave:
    - FeedForward: Bases de las redes neuronales densas.
    - CNN (Convolutional Neural Networks): Especialización en Visión por Computador.
    - RNN (Recurrent Neural Networks): Modelado de secuencias y Series Temporales.
